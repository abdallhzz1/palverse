# Palverse Public UI Roadmap

## Phase 1: Core Framework (Current)
- Setup Next.js, Tailwind, and Environment.
- Implement Light/Dark mode foundation and RTL localization.
- Build Header, Footer, and responsive Layouts.
- Integrate Brand Assets (Logos, Illustrations, Patterns).
- Establish API Client structure.

## Phase 2: Domain Components & Home Page
- Create reusable Store, Category, and Offer cards.
- Implement Search and filtering UI.
- Build the Home Page (Hero, Categories, Featured Stores, Newest Stores).
- Add Mobile Navigation Drawer and Bottom Nav.

## Phase 3: Detail Pages & Routing
- Implement Store Detail page (Cover, Info, Gallery, Working Hours).
- Implement list pages (Stores, Categories).
- Add Static Pages and FAQs.

## Phase 4: Polish & Performance
- Full accessibility check.
- SEO improvements and metadata generation.
- Performance profiling and asset optimization.
- Verify exact fidelity with references across multiple viewports.
