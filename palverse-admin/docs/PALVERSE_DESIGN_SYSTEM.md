# Palverse Design System

## Brand Colors

- **Primary Dark Green**: `#0F3D2E`
- **Primary Green**: `#1E7D4E`
- **Muted Green**: `#7FA789`
- **Light Green Background**: `#EAF3EC`
- **White**: `#FFFFFF`
- **Dark Text / Dark Surface**: `#1F2522`

## Semantic Tokens

- `--background`: Base application background.
- `--foreground`: Main text color.
- `--primary`: Main action color.
- `--secondary`: Secondary action or subtle accents.
- `--card`: Background of card surfaces.
- `--border`: Standard border color.

## Typography

- **Arabic Base Font**: `Cairo` (next/font/google)
- **Latin/Numeric Font**: `Inter` (next/font/google)

## Theming

Light mode is the default and adheres strictly to the pristine visual identity. Dark mode maintains the core Palverse green tones while shifting backgrounds to deep dark green/gray (`#111714` and `#1F2522`).

## Layout Components

- **Sidebar**: Fixed positioning, dark green background in light mode, visually separated with subtle borders.
- **Cards**: Soft rounded corners (`rounded-xl`), delicate borders, subtle shadows.
- **Buttons**: Prominent use of Primary Green for main actions, with clear hover and focus ring states (green-tinted).
